## نظرة عامة

يساعدك **Freelance Pricing AI** على تقدير أسعار مشاريع البرمجيات والويب بناءً على:

- نوع الخدمة والوصف والمخرجات
- التعقيد التقني والأدوات والميزات
- الجهد المقدر وسرعة التسليم
- خبرة المستقل ومستواه
- موقع السوق (محلي / إقليمي / عالمي)

يُنتج التطبيق **ثلاث باقات** (اقتصادية، قياسية، مميزة) مع تبرير باللغة العربية، ويدعم إنشاء **تقرير PDF** قابل للتخصيص بالشعار والبيانات التواصلية.

## المميزات

| الميزة | الوصف |
|--------|--------|
| **معالج خطوات (Wizard)** | 5 خطوات منظمة: الخدمة → التعقيد → الجهد → الخبرة → السوق |
| **تسعير بالذكاء الاصطناعي** | استخدام نموذج Gemini لصياغة باقات وتبرير التسعير |
| **تقرير PDF** | إنشاء تقرير احترافي مع مقدمة وخاتمة وأقسام مخصصة (مولدة بالـ AI) |
| **تخصيص العلامة التجارية** | اسم الشركة، البريد، الهاتف، الموقع، الشعار (Base64) |
| **سجل التسعيرات** | حفظ آخر 50 تسعيرة في المتصفح مع إمكانية التصدير/الاستيراد JSON |
| **واجهة عربية (RTL)** | خط Tajawal واتجاه من اليمين لليسار |
| **حفظ تلقائي** | حفظ حالة المعالج والإعدادات في `localStorage` |

## التثبيت والتشغيل

### 1. استنساخ المشروع أو تنزيله

```bash
git clone <رابط-المستودع> freelance-pricing-ai
cd freelance-pricing-ai
```

### 2. تثبيت الاعتماديات

```bash
npm install
```

### 3. إعداد متغيرات البيئة

انسخ ملف المثال وأنشئ ملف `.env` في جذر المشروع:

```bash
cp .env.example .env
```

ثم عدّل `.env` وضع مفتاح Gemini:

```
VITE_GEMINI_API_KEY=your_actual_api_key_here
```

### 4. تشغيل وضع التطوير

```bash
npm run dev
```

افتح المتصفح على العنوان الذي يظهر (عادةً `http://localhost:5173`).

### 5. بناء نسخة الإنتاج

```bash
npm run build
```

الملفات الناتجة في مجلد `dist/`.

### 6. معاينة نسخة الإنتاج محلياً

```bash
npm run preview
```

## طريقة الاستخدام

1. **الخطوة 1 — الخدمة:** أدخل نوع الخدمة، الوصف، والمخرجات المتوقعة.
2. **الخطوة 2 — التعقيد:** حدد الأدوات، مستوى التعقيد (بسيط/متوسط/معقد)، والميزات.
3. **الخطوة 3 — الجهد:** عدد الساعات المقدرة وسرعة التسليم (عادي/عاجل).
4. **الخطوة 4 — الخبرة:** سنوات الخبرة، المستوى (مبتدئ/متوسط/خبير)، ومشاريع مشابهة.
5. **الخطوة 5 — السوق:** موقع المستقل والعميل ونوع السوق (محلي/إقليمي/عالمي).
6. **النتائج:** عرض الباقات الثلاث مع إمكانية نسخ النتائج، إنشاء تقرير PDF، أو حفظ التسعيرة في السجل.
7. **التقرير:** من زر "تقرير PDF" يمكنك إدخال اسم العميل والمشروع وملاحظات، وتخصيص العلامة التجارية (شعار، بريد، هاتف، موقع)، ثم توليد وتنزيل الـ PDF.
8. **السجل:** من أيقونة السجل يمكنك عرض التسعيرات السابقة، حذف عنصر، تصدير/استيراد JSON، أو مسح الكل.
